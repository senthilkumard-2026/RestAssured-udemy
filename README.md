# RestAssured-udemy
RestAssured API Automation – Udemy Practice Project

This repository contains API automation test scripts created while learning and practicing REST API automation through a Udemy course. It covers essential Rest Assured concepts and real‑world scenarios to help build a strong foundation in API testing.
📌 What’s Included

REST API testing with Rest Assured
GET, POST, PUT, DELETE request examples
Payload creation using POJOs & JSON files
Response validation with Hamcrest matchers
Logging, request/response specifications
Serialization & Deserialization
Parameterization of tests
API testing framework structure using TestNG
Reusable utility classes
End‑to‑end sample automation flows

🧰 Tech Stack

Java
Rest Assured
TestNG
Maven
JSON / Java POJOs

🎯 Purpose
To practice, refine, and demonstrate skills in API Automation Testing using Rest Assured, following a structured and industry‑ready approach.
